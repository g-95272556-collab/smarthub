Aplikasi Web Progresif (PWA) SK Kiandongo

📲 Cara Pemasangan (PWA)

Aplikasi ini tidak memerlukan storan besar. Pasang terus dari pelayar web anda untuk akses pantas.

💻

PC / Laptop (Windows/Mac)

  1. Buka Chrome / Edge.
  2. Layari URL xbasmarthub.netlify.app.
  3. Klik ikon Install di hujung kanan kotak URL (Address Bar).
  4. Klik Install pada pop-up.
📱

Android (Telefon/Tablet)

  1. Buka Google Chrome.
  2. Layari portal rasmi.
  3. Klik menu tiga titik (⋮) di atas kanan.
  4. Pilih Add to Home screen.
🍎

iPhone / iPad (iOS)

  1. Buka pelayar Safari.
  2. Layari portal rasmi.
  3. Ketik ikon Share (kotak beranak panah atas).
  4. Pilih Add to Home Screen.

🔐 Cara Log Masuk

Log masuk adalah menggunakan sistem keselamatan Single Sign-On (SSO).

Langkah 1: Buka aplikasi SmartSchoolHub yang telah dipasang.

Langkah 2: Pilih log masuk menggunakan akaun rasmi (cth: e-mel MOE DL).

Langkah 3: Selepas berjaya, sistem akan memaparkan Dashboard berdasarkan peranan anda.

👨‍🏫 Panduan Khas: Guru

Fokus utama paparan guru adalah kelancaran tugas harian dan pemantauan aktiviti.

📊 Dashboard Utama

Melihat pengumuman rasmi sekolah terkini dan ringkasan am tugasan semasa sebaik sahaja log masuk.

📅 Modul Takwim

Semak takwim sekolah, jadual mesyuarat, dan acara. Boleh ditapis mengikut bulan untuk perancangan.

🏃 Modul Aktiviti

Isi laporan kehadiran kokurikulum dan baca arahan aktiviti terkini terus dari telefon bimbit.

📚 Modul DSKP

Rujukan pantas Dokumen Standard Kurikulum (DSKP) tanpa perlu memuat turun fail berasingan.

⚙️ Panduan Khas: Pentadbir (Admin)

Akses penyuntingan untuk mengurus kandungan aplikasi secara berpusat (Cloudflare D1).

✏️ Pengurusan Takwim & Aktiviti

Tambah, kemaskini, atau padam data takwim dan program. Perubahan akan terpapar secara langsung kepada semua guru apabila disegerak (sync).

🛠️ Tetapan Konfigurasi

Kawal tetapan notifikasi (`TAKWIM_GURU_NOTIF_*`) dan data kritikal lain melalui panel Admin.

Bantuan Teknikal: Sila muat semula (Refresh) jika data tidak dikemas kini.
