# Sample Data

Folder ini mengandungi contoh data yang selamat untuk pembangunan, demo, atau semakan struktur fail.

Prinsip:

- jangan letak data sebenar guru, murid, penjaga, atau dokumen sekolah sensitif di sini
- guna nilai contoh, nama dummy, dan nombor pengenalan palsu
- jika perlu data sebenar untuk kerja operasi, simpan dalam folder tempatan yang diignore Git

Cadangan folder tempatan yang tidak ditrack:

- `private-data/`
- `exports/`
- `working-docs/`

Fail contoh semasa:

- `data_guru.sample.csv`
- `data_murid.sample.csv`
- `data_harilahir.sample.csv`
